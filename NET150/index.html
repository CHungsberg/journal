LAB 1-1

ifconfig – show info about the computer’s IP address information

ping – sends a packet to a given ip and reports how long it takes

tracert – traces a route that a packet takes through the network to its destination

nslookup – looks up the IP address of a domain

On windows you can open the command prompt by pressing Win+R and typing cmd and pressing ENTER

LAB 2-1

To deploy workstations and switches in packet tracer you just select them from the new device menu

They can be connected with cables from the same menu, and you connect them by clicking both devices to create the connection.

To access the command-line terminal on a Packet Tracer PC, you go into the PC’s GUI and select “command line”

LAB 2-2

Wireshark is a useful tool for discovering what network information a computer is transmitting, and can be used for reading the information sent in each packet by the computer

LAB 3-1

dumping the ARP cache in windows allows wireshark to catch the new ARP requests being sent by the computer

LAB 3-2

The Address Resolution Protocol (ARP) is a communication protocol used for discovering the link layer address, such as a MAC address, associated with a given internet layer address, typically an IPv4 address.

LAB 4-1

To set the gateway address on a workstation in Packet Tracer you can navigate to the GUI menu and go to the configure IP menu.
